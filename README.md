# prism-lcd-driver

**Built with:** Rust

## Getting Started

Clone the repository:

```bash
git clone https://github.com/suhteevah/prism-lcd-driver.git
cd prism-lcd-driver
```

## Installation

```bash
cargo build --release
cargo run
```
